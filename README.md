# CANSCANS

CANSCANS is an educational mobile web game designed for toddlers and babies to learn the four core concepts its name stands for: **C**olor, **A**lphabet, **N**umber, and **S**hape. Each category offers three interactive activities — a tap-based game, a flashcard set, and a reference chart — and every element reads aloud when tapped or clicked, making it accessible to pre-readers. The app is built with vanilla HTML, CSS, and JavaScript (with jQuery) and is fully optimized for mobile browsers.

Beyond the four learning modules, CANSCANS includes a **Magnetic Board** feature that lets children (and parents) create unlimited customizable blocks in any combination of color, letter, number, and shape — mimicking a real magnetic letter-board experience in the browser. The project is hosted on GitHub Pages and requires no installation; simply open the live link below in any modern mobile or desktop browser to start playing.
