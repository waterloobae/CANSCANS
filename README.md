# CANSCANS

CANSCANS is an interactive educational web app designed to help toddlers and young children learn the fundamentals of **C**olours, **A**lphabet, **N**umbers, and **S**hapes. Each of the four learning modules offers three engaging activity modes — an interactive game, illustrated flashcards, and a visual reference chart — all paired with audio pronunciation to reinforce auditory learning. The app also features a creative **magnetic board** where children can freely drag, drop, and arrange colourful shapes and letters on a canvas, encouraging open-ended play and fine motor skill development.

Built entirely with HTML5, CSS3, and JavaScript (using jQuery and the Canvas API), CANSCANS runs as a fully static, mobile-friendly web application with no installation required. It is optimised for touch-screen devices and supports Progressive Web App (PWA) capabilities so it can be added to a home screen for a native app-like experience. Try it live at **https://waterloobae.github.io/CANSCANS/index.html**.
