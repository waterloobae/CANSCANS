# CANSCANS

CANSCANS is an educational mobile web game designed for toddlers and babies to learn four foundational concepts: **C**olour, **A**lphabet, **N**umber, and **S**hape. Each of the four learning modules offers three interactive activity types — a game, flashcards, and a reference chart — supported by audio pronunciations and colourful visuals to engage young learners. The app is built with HTML5, JavaScript (jQuery), the Web Audio API, and the Canvas API, and is optimised for touch-based mobile devices.

In addition to the four core modules, CANSCANS includes a **Magnetic Board** where children (and parents) can freely drag and arrange customisable blocks in different colours, letters, numbers, and shapes. The entire application runs in the browser with no installation required and is available live at https://waterloobae.github.io/CANSCANS/index.html.
